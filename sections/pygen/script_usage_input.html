Generate Sections from the Python script

This documentation allows you to understand how to use the create_section_input.py script, to generate a section or group, and its respective HTML document based on a dummy template.

Script API Documentation

Introduction

This script is designed to create sections and groups of HTML content based on user input. It efficiently handles the creation and management of unique section and group IDs, as well as the generation of HTML files for each section.

Files Involved
File Name Description
sections.json A JSON file containing the structure of sections and groups, including their unique IDs, labels, and URLs.
create_section_input.py The main Python script that processes user input and updates 'sections.json' and the corresponding HTML files.
Structure

The script takes user input to create sections or groups. Sections contain HTML content and are stored in separate files, while groups can contain other sections or groups, forming a nested structure.

Usage

Run the 'main_script.py' and follow the prompts to input the required information. The script will then generate the appropriate section or group, update the 'sections.json' file, and create or update the corresponding HTML files.

Input Example

Section type (1="section", 2="group"): 1
Section/Group label: My Section
Section ID/Group ID (unique): my_section
Parent Group ID (0 for none): 0
Section Content (based on provided example): [{"type":"t","v":[1,"Header Example"]},{"type":"p","v":"Paragraph Example."}]

* Note the Section Content is provided in a single line.

Input Requirements

When running the script, you'll be prompted to provide the following inputs. Each input must be provided in one line:

Input Name Description Example
Section Type An integer (1 or 2) indicating if it's a section (1) or group (2). 1
Section/Group label A string representing the label for the section or group. My Section
Section/Group ID A unique string identifier for the section or group. my_section
Parent Group ID A string indicating the parent group ID. Use '0' if there's no parent group. 0
Section Content An array with the required format in 'Object Types'. See example in the code block below this table.

Section Content Example

Note: When entering the 'Section content' input, the content must be provided as a single line JSON string to avoid errors in the script.

[
    {
        "type":"t",
        "v": [1,"Header h1 Example"]
    },
    {
        "type":"p",
        "v":"Parragraph Example."
    },
    {
        "type":"s",
        "v":"{
            'Code': 'Block',
            'v': [
                5,
                "Your <b>SNIPPET<br> Example<>"
            ]
        }
    },
    {
        "type":"ta",
        "v":{
            "h":["Header 1", "Header 2"],
            "r":
            [
                ["Row 1 Col 1", "Row 1 Col 2"],
                ["Row 2 Col 1", "Row 2 Col 2"]
            ]
        }
    }
]

Section Content - Object Types

Object Type Description
t Title:
contains an integer and a string as an array in 'v'.
The integer represents the header level (1-5), and the string is the header text.
p Paragraph:
contains a string in 'v' representing the paragraph content.
s Code snippet:
contains a string in 'v' representing the code snippet content.
ta Table object:
contains a dictionary in 'v' with two keys:
'h' for table headers (list of strings)
'r' for table rows (list of lists of strings).

Conclusion

This script provides an efficient way to create and manage sections and groups with unique IDs, generate corresponding HTML files, and store the structure in a JSON file.
