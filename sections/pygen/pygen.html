Generate Sections from a Python script

This is an example of how to use the generate_html() method in the pyGenerator/generator.py script to generate an HTML document template based on a dummy template.

Modify the input_arr providing a valid array based on the format below.

Possible objects for the array may include:

Name Type (Type) v (Value) Description
title t Array [HeaderSize(Number),Text(String)] Generates a title with the header size specified and with the text specified.
Parragraph p Text(String) Generates a parragraph with the text specified.
table ta Array [h(Array),r(Array[Array])] Generates a table with headers, and rows.
Snippet s Text(String) Generates a block code snippet.
Object Title example:
{
    'type': 't',
    'v': [
        5,
        "Your <b>title<br> here</b>"
    ]
}
Object Title Key Value list:
Name Type Description
[0] Header Size Number From 1 to 5. Is the header size.
[1] Header Text String The text inside the element, can content valid HTML tags, such as <b>Mi<br>Title</b>
Object Parragraph example:
{
    'type': 'p',
    'v': "Your <b>parragraph<br>text</b>here."
}

In this case the key 'v' is directly the string of the parragraph, which can also include valid HTML Tags as the title.

Object Table example:
{
    'type': 'ta',
    'v': {
        'h': ['Header1', 'Header2'],
        'r': [
            ['Row1 Col1', 'Row1 Col2'],
            ['Row2 Col1', 'Row2 Col2']
        ]
    }
}
Object Table Key Value list:
Name Type Description
h Headers Array An array of strings, each representing a header for the table.
r Rows Array[Array] An array of arrays, where each inner array represents a table row with its respective cell values.
Object Snippet example:
{
    'type': 's',
    'v': "def hello_world():\n    print('Hello, World!')"
}

In this case the key 'v' is directly the string of the parragraph, which can also include valid HTML Tags as the title.
