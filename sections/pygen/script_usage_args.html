Generate Sections from the Python script

This documentation allows you to understand how to use the create_section_args.py script, to generate a section or group, and its respective HTML document based on a dummy template.

Structure of the JSON Content

The JSON content should follow this format:

[
{"type":"t","v":[1,"Header h1 Example"]},
{"type":"p","v":"Paragraph Example."},
{"type":"s","v":"{\n    'Code': 'Block',\n    'v': [\n        5,\n        \"Your <b>SNIPPET<br> Example</b>\"\n    ]\n}"},
{"type":"ta","v":{"h":["Header 1", "Header 2"],"r":[["Row 1 Col 1", "Row 1 Col 2"],["Row 2 Col 1", "Row 2 Col 2"]]}}
]

Calling the Script

To call the script, use the following command:

python create_section_args.py -t 1 -l "My Section" -i my_section -g 0 -c "your_content_file.json"

Parameters

List of parameters:

Parameter Description Example
-t Type:
An integer (1 or 2) indicating if it's a section (1) or group (2).		 1
-l Section/Group Label:
A string representing the label for the section or group.		 My Section
-i Section/Group ID
A unique string identifier for the section or group (it will be the name of the file).		 my_section
-g Parent Group ID
A string indicating the parent group ID. Use '0' if there's no parent group (the file will be saved on this folder).		 0
-c Section Content
An array of objects with the required format in 'Object Types'.		 See example in the code block below this table.

Section Content - Object Types

Type Description Example
t Title. Must contain in 'v' an array with: an integer (1 - 5) and a string text. {"type":"t","v":[1,"Header h1 Example"]}
p Paragraph. Must containt in 'v' and a string text. {"type":"p","v":"Paragraph Example."}
s Code Block Snippet. contains a string text in 'v' representing the code snippet content. {"type":"s","v":"{\n 'Code': 'Block',\n 'v': [\n 5,\n "Your SNIPPET
Example"\n ]\n}"}
ta Table. contains a dictionary in 'v' with two keys: 'h' for headers (list of strings) and 'r' for rows (list of lists of strings) {"type":"ta","v":{"h":["Header 1", "Header 2"],"r":[["Row 1 Col 1", "Row 1 Col 2"],["Row 2 Col 1", "Row 2 Col 2"]]}}
