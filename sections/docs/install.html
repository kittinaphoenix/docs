Installation

To install the docs, follow these steps:

1.Clone the repository using the following command:

git clone https://github.com/kittinaphoenix/docs.git

2.Change directory into the repository:

cd gptDetectives-docs

3.Open the repository in your browser:

open index.html

Usage & Test

Enabling CORS in Chrome

If you're running the project locally you might encounter a CORS (Cross-Origin Resource Sharing) error. To enable CORS in Chrome, follow these steps:

Create a new shortcut for Chrome on your desktop or in a folder.

Right-click the new shortcut and select Properties.

In the Target field, add the following at the end of the path after chrome.exe: --disable-web-security --user-data-dir=c:\temp

Click OK to save the changes.

Open the new shortcut and you should now be able to access resources from other domains without encountering CORS errors.

Note that this should only be used for local development purposes and not for browsing the web in general.

That's it! You now have the GPTDetectives documentation installed on your local machine.
